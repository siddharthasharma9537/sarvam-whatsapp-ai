# Sarvam WhatsApp API

Sarvam is a FastAPI-based WhatsApp automation backend for Sri Someshwara Shivagnana Peetham.

It handles:
- Devotee registration
- Temple history delivery
- Next Tithi lookup
- Language selection (English / Telugu)
- MongoDB persistence
- WhatsApp Cloud API integration
